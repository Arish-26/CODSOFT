Student Grade Calculator (Java)

This project is a Student Grade Calculator developed as part of the CodSoft Java Internship Task.
The program calculates the total marks, average percentage, and grade based on marks entered for multiple subjects.

Features

Takes number of subjects as input

Accepts marks for each subject

Calculates total marks

Calculates average percentage

Displays grade based on percentage

Console-based Java application

Technologies Used

Java

Scanner class

Loops and conditional statements

How the Program Works

User enters the number of subjects

User enters marks for each subject

Program calculates:

Total marks

Average percentage

Grade based on conditions

Grade Criteria

90% and above → O

80% – 89% → A+

70% – 79% → A

60% – 69% → B+

50% – 59% → B

Below 50% → No Grade

Sample Output
Enter Number of Subjects:
5
Enter mark for Subject 1:
85
Enter mark for Subject 2:
90
Enter mark for Subject 3:
78
Enter mark for Subject 4:
88
Enter mark for Subject 5:
92

Result:
Total marks Obtained : 433/500
Average Percentage : 86.6 %
Grade Obtained : A+

How to Run

Install Java (JDK)

Compile the program:

javac Main.java


Run the program:

java Main
